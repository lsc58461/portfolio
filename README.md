## 프로젝트 주요 기능

- 연락 수단 제공: 사용자와의 연락을 쉽게 할 수 있는 수단을 제공합니다.
- 보유 기술 스택: 보유 중인 기술 스택을 방문자들에게 제공합니다.
- 포트폴리오 소개: 사용자의 프로젝트 및 경험을 소개하는 페이지를 제공합니다.
